# Lung Cancer Prediction using Machine Learning

This project uses Machine Learning techniques to predict the risk of lung cancer based on patient health parameters such as smoking habits, anxiety, fatigue, and chronic disease. The model analyzes the dataset and predicts whether a patient is likely to have lung cancer.# lung-cancer-prediction-ml
Machine Learning model to predict lung cancer using patient health data.
